# MokXya Hisab Post-Deployment Checklist

This checklist contains owner-verifiable items to confirm the successful release of the MokXya Hisab pre-launch website.

- [ ] Cloudflare deployment shows Success on the dashboard (Owner Action Required)
- [ ] Deployed commit matches GitHub main branch
- [ ] Homepage opens at `https://mokxya.pages.dev`
- [ ] Direct routes (`/about`, `/pilot`, `/privacy`, `/terms`) open properly
- [ ] Mobile menu works on narrow devices
- [ ] Pilot mode is correct (currently set to endpoint test mode)
- [ ] Favicon appears in the browser tab
- [ ] Open Graph image loads when shared
- [ ] Canonical URL is correctly pointing to `https://mokxya.pages.dev`
- [ ] `robots.txt` opens
- [ ] `sitemap.xml` opens
- [ ] No old pages or boilerplate content remains cached
- [ ] No broken images
- [ ] No console errors
- [ ] Social preview should be tested in Twitter/LinkedIn validators
- [ ] Legal copy should receive professional review
- [ ] Pilot contact channel should be configured when ready
- [ ] Custom domain can be connected later
- [ ] Metadata must be updated when the custom domain changes
